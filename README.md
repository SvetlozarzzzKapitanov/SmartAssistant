# SmartAssistant

Smart Assistant мобилно приложение за планиране и вземане на решения, базирано на време, задачи и локация.

## Основна идея
Приложението подпомага потребителя да:
- организира задачи и график
- избира кога да тръгне
- оптимизира ежедневни решения

## Features (in progress)
- Notes / To-do
- Scheduler (one-time + recurring)
- Location-based tasks
- Google Maps integration
- "When to leave" logic

## Tech Stack
- Kotlin
- Android SDK
- Google Maps APIs

## Цел
Да се създаде context-aware planning приложение, което комбинира различни източници на информация за по-добро вземане на решения.
